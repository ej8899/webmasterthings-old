How To Install:

Simply upload the script to your server. The create a HTML page, as defined below with the appropriate information in;

There are several HIDDEN variables that need to be defined. These are;

subject - This is the subject you want to be sent out. This does not have to be hidden, but if you want control over what the subject is, use a hidden tag Example HTML code;

<input type="hidden" name="subject" value="www@youradds.com">

count - This needs to be a numerical value (e.g 1,2,3 or 4). This will then define how many people you are trying to send emails out to. Example HTML code;

<input type="hidden" name="counter" value="2">

message - This is obviously the message that you want sent out in the emails. Again, as with the subject is does not need to be a hidden tag, but you may want it as one so you can control what is sent out. Example HTML code;

<input type="hidden" name="message" value="my message can go here">

adminemail - This is your email address. It will be used as the From: part of the email. Example HTML code;

<input type="hidden" name="admin" value="webmaster@yoursssite.com">


There are also a couple of text boxes that need to exist for it to work correctly. These are identified below;

person[x] - This is the name of the person they are refering. For example, if you had the option to send to 2 people, you would have one box called person[1], and another called person[2]. Example HTML code;

<input type="text" name="person[1]" size="20">

email[x] - This is the email of the person they are refering. For example, if you had the option to send to 2 people, you would have one box called email[1], and another called email[2]. Example HTML code;

<input type="text" name="email[1]" size="20">
