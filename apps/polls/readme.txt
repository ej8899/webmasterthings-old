Requirements:

  - MySQL 3.22.x or higher
  - PHP3 or PHP4

Installation:

1. Open the configuration file 'config.inc.php' with a text editor
   and set up your database settings.

   $dbName       = "YourDBName";
   $sql_hostname = "localhost";
   $sql_username = "YourUsername";
   $sql_password = "YourPassword";

2. Set the permissions for all *.php files to mode 755.

3. Call the install script from your browser -> http://www.yourURL.com/poll/install.php
